<h1>checktrace</h1>

This simple Bash script will check whether the HTTP <b>"TRACE"</b> method is enable on the target Urls.

<h2>- Prerequisites</h2>
1) Save the Target Url(s) in a file name <b>"allurls.txt"</b><br>
2) Keep the bash script <b>"checktrace.sh"</b> and <b>"allurls.txt"</b> in the same directory.<br>
